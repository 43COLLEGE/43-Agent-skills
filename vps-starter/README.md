# vps-starter — VPS 小白起步陪跑

> 43 COLLEGE 凯寓出品

第一次买了 VPS 不知道怎么下手？这个 skill 带你从"收到服务器账号邮件"走到"一台干净、安全、可用的 Linux 服务器"。全程自然语言交互，不用提前学命令行。

## 适合谁

- 刚买了第一台 VPS，登录那一步就卡住的
- 看教程满屏命令头大、不知道哪步该做哪步不该做的
- 想给长辈、同事、家人配个远程服务器，又怕他们搞坏的
- 跟着视频教程跑过一次但不记得为什么那样做的

## 不做什么

这个 skill **只打地基**，不教具体应用。装完之后你要：
- 搭 WireGuard / OpenVPN 做远程访问、家里 NAS 回连、公共 WiFi 加密 → 参考官方文档（`wg-easy` 是对小白较友好的 Docker 面板）
- 搭博客、建站、跑 Docker → 到这一步你已经不是小白了，可以找对应应用的独立教程

**明确不做**：不引导任何绕过网络审查的代理工具。想做这类，请自己看相应项目的官方文档。

## 怎么用

打开 Claude Code，复制下面任一句发给 AI：

```
我买了个 VPS 不会用
```

```
第一次用服务器，带我走一遍
```

```
/vps-starter
```

AI 会先问你三个问题（电脑是 Mac 还是 Windows、VPS 买了没、有没有登录信息），然后一步步带你走。每一步做完你贴反馈，它确认无误再进下一步。

## 全流程预览

1. **选 VPS**（还没买时）—— 教你判断一款 VPS 够不够用，不绑定任何服务商
2. **SSH 首次登录** —— Mac 用 Terminal，Windows 用 PowerShell 或 Termius，分开讲
3. **系统初始化** —— 更新系统、建普通用户、配 SSH 密钥、加固 sshd
4. **防火墙** —— ufw 基础配置，只开必要端口
5. **交接** —— 留几个常用命令，告诉你哪些东西要妥善保管

跑完大概 20-40 分钟（取决于网速和熟练度）。

## 为什么不直接给一键脚本

一键脚本省时间，但出了问题你不知道哪里错。这个 skill 的目标不是"最快让你有台服务器"，是"让你理解这台服务器是怎么回事"。下次再遇到问题，不用每次都找人。

## 跨平台

- **macOS**：原生 Terminal + 内置 `ssh`，零额外安装
- **Windows 10/11**：PowerShell 已内置 `ssh`；或用 Termius / MobaXterm 图形化客户端
- **Linux**：本身就是 Linux，不用说了
